<h2 align="left">📖 Introduction</h2>

###

<p align="left">This university project was developed as part of a Data Mining course.<br><br>The project focuses on predicting whether a patient is infected with COVID-19 based on various health and medical attributes using machine learning techniques. It demonstrates the complete data mining workflow, from data preprocessing and feature selection to model training and evaluation.</p>

###

---

<h2 align="left">🔍 Overview</h2>

###

<p align="left">COVID-19 diagnosis can be supported by analyzing patient health indicators and applying predictive machine learning models.<br><br>In this project, several classification algorithms were trained and compared to determine which model achieved the highest predictive performance. The project was implemented using Orange Data Mining, which provides a visual environment for data preprocessing, model building, and evaluation.</p>

###

---

<h2 align="left">🎯 Objectives</h2>

###

<p align="left"><b>The main objectives of this university project were:</b><br>
  ✅ To preprocess and clean the dataset.<br>
  ✅ To handle missing values and detect outliers.<br>
  ✅ To perform feature selection and dimensionality reduction.<br>
  ✅ To train multiple machine learning models.<br>
  ✅ To evaluate and compare model performance.<br>
  ✅ To identify the best-performing model for COVID-19 prediction.</p>

<h2 align="left">📂 Dataset</h2>

###

- The dataset contains patient health and medical attributes used to predict whether the patient is COVID-19 positive or negative.
- It consists of 5,000 records and 18 columns, providing a solid foundation for training and testing models.
- The dataset was selected from the <a href="https://www.kaggle.com">Kaggle</a> website.<br>
- Link of Dataset: <a href="https://www.kaggle.com/datasets/miadul/covid-19-patient-symptoms-and-diagnosis-dataset">COVID-19 Dataset</a><br></p>
